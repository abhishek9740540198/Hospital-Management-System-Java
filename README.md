# Hospital Management System (Java)

A console-based Hospital Management System developed using Java, OOP, ArrayList, and Scanner.

## Features
- Add Patient
- View Patients
- Add Doctor
- View Doctors
- Book Appointment
- View Appointments

## Technologies Used
- Java
- OOP
- ArrayList
- Scanner

## How to Run

Compile:
javac *.java

Run:
java HospitalManagementSystem
